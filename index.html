my resume

projects

Aurora

A simple website using Google's API (Places API, Geocode API, Maps) along with OpenWeather API to determine the day and weekly forecast for a certain location

DataOrganizer

This simple program is used for organzing existing .csv files and sort them based on certain columns, also supports multiple sorting priority, as well as certain rows either by ascending or descending order.

Random Password Generator

A website to generate a random password with a desired length along with options such as including numerical, special, and uppercase characters. There is also an option to not include certain characters. Made with HTML, CSS and Javascript

Restaurant Inspection Manager

This Android app allowed a user to search for a restaurant and display information about it such as the last inspection date, whether it passed or not, and where it is on the map. Features like being able to favorite a restaurant, or clustering nearby restaurants with also support for Spanish language.

Depth of Field Calculator

Android app that included features such as calculating the depth of field of a camera by inputting values, the ability to edit, delete, and add lenses with icons which are saved even through app restarts.

Cure The Virus

Android app game similar to minesweeper but simpler. Try to find all viruses with the fewest moves possible. Hints are given on how many viruses are nearby when an area is clicked on.

Reversi

For the final project of our CMPT 135 class we were to recreate the classic board game "Reversi" in C++. I designed a simple and clean user interface. I implemented my AI to prioritize certain moves and calculate the current best move that would result in the most flipped pieces.

Tic-Tac-Toe

As I was new to C++, I wanted to create simple beginner games and Tic-Tac-Toe was one of them. It is played with two players played on the terminal.

Connect-4

Another simple game I recreated was Connect 4. The game is played with two players on a terminal which displays colored pieces to easily distinguish your opponents and your pieces.
