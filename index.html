my resume

projects

Binary Search Tree

For the final project of our CMPT 225 class we were to implement a pointer based binary search tree to store and retrieve customer data. The main focus was designing a binary search tree along with object oriented programming approach such as classes and encapsulation.

Reversi

For the final project of our CMPT 135 class we were to recreate the classic board game "Reversi" in C++. I designed a simple and clean user interface. I implemented my AI to prioritize certain moves and calculate the current best move that would result in the most flipped pieces.

Tic-Tac-Toe

As I was new to C++, I wanted to create simple beginner games and Tic-Tac-Toe was one of them. It is played with two players played on the terminal.

Connect-4

Another simple game I recreated was Connect 4. The game is played with two players on a terminal which displays colored pieces to easily distinguish your opponents and your pieces.
